Общие требования:

Для UI можно использовать любой фреймворк (в порядке приоритета: WPF, WinForms, консольное приложение).
Результатом выполнения задания является исходный программный код в виде проекта для Visual Studio. Результат работы нужно оформить в виде ZIP архива и сопроводить README файлом. Не включай в архив папки с бинарными файлами (папки bin, obj и `packages`). Также, проект можно выложить на GitHub.
Приложения должны иметь законченный вид и предоставлять удобный пользовательский интерфейс.
